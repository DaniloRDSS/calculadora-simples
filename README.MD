# Calculadora em Java

  

Esse é um codigo simples e basico com funções basicas de somar, subtrair, multiplicar e dividir na linguagem java.

Criado para praticar os fundamentos da linguagem java, e utilizando da função Scanner para entrtada de dados no programa para realização das operações.

  

# Funcionalidades

  

As funcionalidades dessa calculadora basica são:

- Soma

- Subtração

- Multiplicação

- Divisão

  

# Estrutura

  

A estrutura é simples com dois arquivos, o calc.java dentro da pasta código, e na raiz o arquivo de descrição.

  
  
```
|  codigo/

|   |

|   |----  calc.java  # condendo o codigo do programa.

|--------  README.MD   # Arquivo de descrição
```


  
  

# O que preciso para executar o código

  

Você precisara instalar o Java. Pra verificar se você pouse o Java instalado, abra o cmd do Windows e digite na janela que abrir "java --version", isso retornará a versão atual do Java, caso o seu não apareça algo como:

```
Java 24.0.2 2025-07-15
Java(TM) SE Runtime Environment (build 24.0.2+12-54)
Java HotSpot(TM) 64-Bit Server VM (build 24.0.2+12-54, mixed mode, sharing)
```
  

Voce precisará instalar o Java primeiro. 
  

# Exemplo de Uso
```
-----CALCULADORA-----

Digite o primeiro valor para realizar a operação: 10

Digite agora o segundo valor para realizar a operação: 5

  

Informe a operação desejada:

1 - Somar

2 - Subtrair

3 - Multiplicar

4 - Dividir

  

3

A multiplicação entre os valores é: 50.0
```
